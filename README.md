
## 本地开发指南

如果你想在本地运行或预览这个网站，请按照以下步骤操作：

### 1. 克隆仓库

```bash
git clone [https://github.com/nebesn1/navaid-site.git](https://github.com/nebesn1/navaid-site.git)
cd navaid-site
```

### 2. 安装依赖

请确保本地已安装 [Node.js](https://nodejs.org/) (建议 LTS 版本)。

```bash
npm install
```

### 3. 构建静态文件

```bash
npm run build
```

生成的静态文件将会存放在 _site (或你配置的输出) 目录下。

### 4. 本地启动开发服务器

```bash
npm run start
```

启动成功后，在浏览器访问 http://localhost:8080 即可实时预览修改（支持热更新）。

## Local Development Guide

If you want to run or preview this website locally, follow these steps:

### 1. Clone the repository

```bash
git clone [https://github.com/nebesn1/navaid-site.git](https://github.com/nebesn1/navaid-site.git)
cd navaid-site
```

### 2. Install Dependencies

Make sure [Node.js](https://nodejs.org/) is installed locally (an LTS version is recommended).

```bash
npm install
```

### 3. Build Static Files

```bash
npm run build
```

The generated static files will be stored in the _site directory (or the output directory you configured).

### 4. Start the local development server

```bash
npm run start
```

Once the server is up and running, visit http://localhost:8080 in your browser to preview your changes in real time (hot reload is supported).
